1. Элементом называется любая часть вебстраницы. Обычно представляет из себя открывающий тег с рядом атрибутов или
без них, содержимое и закрывающий тег.
2. Атрибуты - специальные слова которые помогают управлять поведением HTML элемента.
3. Семантическая верстка - такой подход к разметке , который опирается не на содержимое сайта, а на смысловое предназначение каждого
блока. Семантическая верстка очень важно , она помогает поднимать сайт выше в поиске , так как посковые роботы 
делают приоритетными те сайты , которые понятны для их алгоритмов. Так же семантика важна для скринридеров, это 
специальные программы для людей с ограничеными возможностями которые зачитывают содержимое сайта вслух.
4. IFrame — это кадр внутри кадра. Это компонент HTML-элемента, который позволяет встраивать документы, видео и 
интерактивные медиафайлы на страницу. То есть простыми словами помогает встраивать мультимедиа в страницу.
5. URL кодирование в HTML. URL может содержать только символы из набора ASCII и поскольку часто URL часто содержат символы
которые не входят в набор ASCII - адрес может быть приобразован в правильный формат ASCII. Кодирование URL заменяет опасные символы ASCII на "%",
за которыми следуют две шестнадцятеричные цифры.URL-адреса не могут содержать пробелы. 
Кодирование URL обычно заменяет пробел знаком плюс (+) или %20.
6. Формы. Для получения каких либо данных от пользователя используются формы. Форма создается с помощью тега form.
Внутри мы можем использовать различные теги для получения нужной информации от пользователя. При отправке формы 
по дефолту происходит обновление страницы , убрать это поведение можно с помощью event.preventDefault(). Так же
Мы можем получить доступ к данным которые ввел пользователь которые будут представлены в виде объекта.
7. Canvas & SVG. SVG - векторная графика, позволяет очень гибко работать с изображениями , чаще всего используется для иконок.
Позволяет гибко менять размер изображения , а так же цвет заливки , обводки и тд.
Canvas - html элемент который позволяет работать с графикой. чаще всего используется для отрисовки графиков ,
для работы например с рисованием.
8. Audio и video. В html работа с видео выполняется с помощью тега <video>. Можно отрисовывать как статическое видео, 
так например и делать стрим с вебкамеры.
Работа с аудио ведется с помощью тега audio.
